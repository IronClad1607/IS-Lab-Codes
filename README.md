# IS-Lab-Codes

This repository has various cryptography codes. The following techniques are involved:

- Caesar Cipher
- Poly Alphabetic Cipher
- Transpositional Cipher
- Affine Cipher
- Diffie Hellman Algorithm
- Play Fair Cipher
- RSA Algorithm
- Rail Fence Cipher
- Vernam Cipher
- Hill Cipher
- Auto Key Cipher
- Euclidean Algorithm
- DES Algorithm
- Multiplicative Cipher
